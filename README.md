<h5>Getting started</h5>
<small>
<p>To get the Node server running locally:</p>
<ul>
  <li>Clone this repo</li>
  <li>
    Install all required dependencies

    npm install

  </li>
  <li>
  Create <strong>.env</strong> file, copy from <strong>.env.example</strong> file
  </li>
  <li>
    Run app

    npm start

  </li>
</ul>
</small>
